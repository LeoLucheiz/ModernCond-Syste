Projeto: ModernCond

Sistema de gestão de condomínios em C#. Atualmente o sistema já conta com gerenciamento de moradores e histórico de mensalidades.

O que já funciona:

Cadastro de histórico (Pagamentos/Pendências).

Cálculo automático de dívida por morador.

Filtros de inadimplência usando LINQ.

Próximos passos (Em desenvolvimento):

Menu interativo com Switch Case.

Opção de dar baixa em pagamentos.

Adição/Remoção de moradores via console.
